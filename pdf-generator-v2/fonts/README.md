# PDF Generator V2 - Fonts

Bu klasör WeasyPrint için kullanılacak fontları içerir.

## Otomatik İndirilen Fontlar
Docker build sırasında Inter font ailesi otomatik olarak indirilir:
- Inter-Regular.ttf
- Inter-Medium.ttf
- Inter-Bold.ttf
- Inter-Black.ttf

## Manuel Font Ekleme
Ek fontlar eklemek isterseniz `.ttf` formatında bu klasöre kopyalayın.

## Desteklenen Formatlar
- TrueType (.ttf)
- OpenType (.otf)
- WOFF (.woff) - sınırlı destek
